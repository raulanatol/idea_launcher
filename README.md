# Idea launcher

It will open your idea project from your terminal.

## Install

```shell
sudo make install
```

## Uninstall

```shell
sudo make uninstall
```

## Usage

### IntelliJ Idea

In your project directory:

```shell
idea
```

### Webstorm

In your webstorm project directory:

```shell
webstorm
```
